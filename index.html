How I Built a Responsive Website Without Any Frameworks

By John Doe June 1, 2023 8 min read

Building a responsive website without relying on frameworks like Bootstrap or libraries like jQuery is not only possible but can result in faster, leaner websites. Here's how I did it...

The Approach

I started with a mobile-first approach, writing CSS that would work well on small screens and then adding media queries to adapt the layout for larger screens.

The HTML Structure

Semantic HTML5 elements like header, nav, main, article, and footer provide good structure and accessibility out of the box.

CSS Techniques

I used modern CSS features like Flexbox and Grid for layout, CSS variables for theming, and relative units (em, rem, %) for responsive sizing.

JavaScript Considerations

Modern vanilla JavaScript can handle most interactive needs without jQuery. I used classList for DOM manipulation and fetch() for AJAX requests.

Performance Results

The framework-free approach resulted in a 95% Lighthouse performance score with just 15KB of CSS and 8KB of JavaScript.

Conclusion

While frameworks have their place, building without them can lead to more performant, focused websites when done properly.
